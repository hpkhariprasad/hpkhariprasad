# Third-party code

`api/`, `src/`, and `themes/` are vendored from
[anuraghazra/github-readme-stats](https://github.com/anuraghazra/github-readme-stats),
used under the MIT License (see `LICENSE-github-readme-stats`). They are copied
here, not forked, so they won't receive upstream updates automatically.

This deployment powers the GitHub stats and pin cards in this profile's
`README.md`, hosted on Vercel with a personal `PAT_1` token to avoid sharing
the public instance's rate limit.
